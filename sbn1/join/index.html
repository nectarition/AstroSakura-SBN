サークル参加条件

  • サークル代表者が義務教育課程を修了していること
  • 成人向け作品を頒布する場合、参加代表者が18歳以上であること
  • TOKYO6キャラクター(小春六花, 夏色花梨, 花隈千冬) を含む頒布物が１つ以上があること

スペースについて

1スペースあたり、机半分(90cm x 60cm)・椅子1脚・サークル通行証2枚が割り当てられます。

1次申込み

  • 1スペース : 4,000円
  • 2スペース : 8,000円

2次申込み

  • 1スペース : 4,500円
  • 2スペース : 9,000円

注意事項

  • 以下に定める物品の販売はできません。
    • 刑法175条に定める「猥褻図画類」を含む物品
    • 古物や法人発行の物品
  • 会場の電源は使用できません。
  • 参加者都合によるキャンセルの場合(当日発熱等で欠席された場合を含む)、原則返金はできません。
  • 最大スペース数を超えた場合、抽選による配置となります。なお、この際の返金は決済手数料を除いた額となります。
  • 紙媒体で本イベントが初頒布の作品は、見本誌として回収いたします。
  • 隣接したいサークルがある場合は、事前にサークル間で打ち合わせを行い、双方ともにサークル名を「運営チームへの連絡事項」に記入して申し込んでください。

申込みの手順

1-１. 申込み情報を入力する

申込みフォームを開き、各種情報の入力をお願いします。

入力の完了後、「送信」を押すとサークル参加費を支払うためのリンクが掲載されたページが表示されます。
申し込んだスペース数のリンクをクリックして、支払いページを開いてください。

郵便振替でお支払いいただく場合は、「郵便振替で支払う」をクリック後、ページの案内に沿ってお支払いいただくようお願いいたします。

1-2. サークル参加費の支払いを行う

金額を確認し、支払いを行ってください。
この際、システムへの申込みに使用したメールアドレスを使用するようにお願いいたします。

1-4. メールボックスを確認し、メールを確認する

仮申込み受付メールと、メールアドレス認証メールが届きます。
仮申込み受付メールは内容を確認し、入力した内容が記載されていることを確認してください。

1-3. メールアドレスの認証を行う

メールが正しく受信されていることを確認するため、メールアドレスの認証を行います。
メールに記載されているリンクをクリックしてください。

認証が正しく完了すると「認証が完了しました」と表示されます。
「認証に失敗しました」と表示された場合は、nctbls@nectarition.jp までお申し付けください。

2-1. サークルカットの提出を行う

サークルカットを提出するには、専用のフォームから別途提出する必要がございます。

Nectarition AIMS(申込み情報管理システム) へアクセスしてください。
ログインボタンからログインページにアクセスし、申込み時に入力したメールアドレスとパスワードを使用してログインしてください。

2-2. 申込み情報照会ページを開く

メニューから「申し込んだイベント」をクリックして、申込み済みイベントページを表示してください。

リストから「しおばな祭 サークル参加」を選択してください。

2-3. 申込みIDを控える

フォーム提出時に「申込みID」をご入力いただきますので、「申込み基本情報」内の「申込みID」をクリップボードにコピーしてください。

2-4. サークルカット提出フォームを開く

ページを下にスクロールすると、サークルカットを提出するためのリンクがございます。
「提出フォーム」をクリックして、フォームを開いてください。

2-5. サークルカットを提出する

「申込みID」に先程コピーしたIDを貼り付けてください。
「サークルカット」に作成いただいた画像データをアップロードしてください。

申し込み確定を確認する

運営チームによるサークルカットの確認、申込み情報の確認が完了すると、確認完了メールが届きます
この２つのメールを受信した時点で申込みが受理され、イベントへの申込みが確定されます。

※「申込み情報確認事項のお知らせ」メールが届いた場合

運営チームによる確認時、申込み情報に不備や確認事項があった場合は上のようなメールが届きます。

＜確認事項＞に記載されている内容を、申込み情報照会ページより確認し、ページ下部の「運営チームへの連絡事項」に必要事項を入力してください。

スペース割り当ての確認方法

スペース割り当てが発表されると、上のようなメールが届きます。

申込み情報照会ページを開き、「スペース割り当て情報」部の「スペース割り当て」にご自身のサークルが配置されたスペース名が記載されております。
この際、申込み時のスペース数と実際に割り当てられたスペース数が異なる場合は、「運営チームへの連絡事項」よりお申し付けください。

システムの都合上、2スペース割り当ての場合は「A-01a, A-01b」といった表記になります。 実際に広報等で用いられる場合は「A-01ab」等の表記で構いません。

郵便振替でお支払いいただく場合

本イベントでは、サークル参加費をお支払いいただく際、クレジットカード等によるオンライン決済の他、現金での郵便振替にも対応しております。
郵便振替は、振替いただいた情報の確認が必要なため、お申し込み確定までお時間をいただく場合がございます。
また、着金締切日は申し込み締切日である3月27日までとさせていただきます。

スペース発表時の流れ

スペースが発表されると、マイページから入場の際に必要な「リストバンド型通行証引き換え証券」が確認できるようになります。

Nectarition AIMS(申込み情報管理システム)にログインし、「マイページ」→「申し込んだチケット」から確認することができます。
1スペース配置の場合は2枚、2スペース配置の場合は4枚、引き換え証券が発行されています。

引き換え証券は、1人1枚ご用意いただくことになりますので、それぞれのチケット詳細ページにある「チケットリンク」のURLを入場される方全員にご送付いただくようお願いいたします。

最大までスペースが配置され、配置漏れとなった場合、１週間以内に返金を行います。
なお、郵便振替を使用された方は、別途ご連絡いたします。

当日の流れ

イベントの開会は11時です。サークル参加者は 9時30分から10時30分までに 入場してください。

入場

  1. チケット詳細ページにある「チケットURL」を開き、下部の「使用される方の情報」に実際に引き換え証券を使用される方の情報を入力してください。
  2. 引き換え証券が使用できる状態になると、画面上部の色が変わり「使用済みにする」ボタンが押せるようになります。
  3. この状態の画面を入口のスタッフまでご提示ください。
  4. スタッフが確認後、「使用済みにする」を押していただき、リストバンド型通行証を受け取ったら入場手続きの完了です。

参加登録･見本誌提出

頒布活動を行うためには、参加登録と見本誌の確認が必要となります。
スペースに到着しましたら、まずは参加登録と見本誌提出のご準備をお願いいたします。

  1. ご自身のスペースに到着しましたら、机上部に貼ってあるスペースシールの確認をお願いいたします。
  2. 机に参加登録･見本誌提出用封筒が置いてありますので、内容を確認し、ご自身のスペース番号、サークル名、新刊数を記載してください。
  3. 封筒に今回頒布する紙媒体の作品を 全て 入れ、運営本部までお持ちください。なおグッズ等は確認の対象外となりますので、ご提出いただかなくて構いません。
  4. なお、成人向けグッズがある場合は確認いたしますので、一緒にお持ちください。

紙媒体で本イベントが初頒布の作品は、見本誌として回収いたします。なお、他のイベントで既に頒布されている既刊等はお返しいたします。

参加登録･見本誌提出が完了しましたら、設営を行ってください。
